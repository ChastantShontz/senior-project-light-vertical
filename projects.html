My Websites

Here are some of the websites I've built on GitHub!

Star Wars API

This website fetches a public Star Wars API and uses it to access info on various characters. A random number generator selects one to be displayed when the page loads, and a pair of buttons allow you to cycle forward or backward through them. (Way too many of these websites feature Star Wars, as you'll soon see!)


URL: https://chastantshontz.github.io/star-wars-api

Tailwind Website

As its incredibly creative name suggests, this Tailwind Website was built using Tailwind CSS. It is fairly simple, as the goal of this project was to successfully install and learn how to work with Tailwind, with no specific goals in mind.


URL: https://chastantshontz.github.io/tailwind-website

Quotes Generator

This Quotes Generator contains an array with some of my favorite Star Wars lines, as well as the characters that said them and links to the sources on YouTube. It can either display a random quote when the page is loaded or cycle through them in chronological order. Each time the quote is changed, it also switches out the color scheme.


URL: https://chastantshontz.github.io/quotes-generator

Hangman

This game of Hangman was the final project for my intro to web development class. It was a gateway for me to experiment with some more advanced JavaScript concepts that I hadn't used before. The petty part of me built this because the previous year, I had gotten 17% on a hangman project in a MATLAB class and wanted to prove that the professor was the problem. This project got a 100%, so I guess my point was made.


URL: https://chastantshontz.github.io/hangman

Storefront

My Storefront was built using a pre-existing template with Bootstrap, albeit heavily modified to fit my vision. It displays the front end of an e-commerce site for an ice cream corporation. The products and sales here are modeled after the ones at Stewart's Shops, where I worked at the time.


URL: https://chastantshontz.github.io/storefront

Rock, Paper, Scissors

Here's the most X-treme game of Rock, Paper, Scissors that you'll ever see! While its design is relatively basic, the main focus was getting the game mechanisms to work with JavaScript. This game can even remember your score from past sessions. I did so well with this project that the following year, when I was a TA for the class, I taught this unit myself.


URL: https://chastantshontz.github.io/rock-paper-scissors

Media Gallery

(Yes, another Star Wars one, I know!) The Media Gallery consists of two main parts: a grid of some of my favorite Star Wars characters and a slideshow of various battles at the top. Each grid item can be expanded to fill the page as a modal, displaying some info about the character in question. This is also the first project that I implemented JavaScript, using it to move the slideshow along.


URL: https://chastantshontz.github.io/media-gallery

Components Library

While itself not a standard website, my Components Library has been very useful in helping me to create other websites. It is essentially a collection of general-purpose HTML and CSS assets that I can copy/paste into other projects (often with a bit of modification). These include two navigation bars, four buttons, four text snippets, and two media cards that take assets from the other categories.


URL: https://chastantshontz.github.io/components-library

Tribute Page

This Tribute Page was the first website I ever built, back when I was a sophomore in college. We got to pick any topic to do a "tribute" to, and I chose my favorite Star Wars character, Grand Admiral Thrawn. This was my introduction to HTML and CSS, although there's no JavaScript in this project as that would come later in the semester. I've since gone back and touched it up quite a bit, but the original design remains the same.


URL: https://chastantshontz.github.io/tribute-page
